# Introduction
blBot is a simple and powerful bot built to make tasks easier and faster.
It helps automate actions, improve workflow, and save time.
Designed with flexibility in mind, blBot is ready to grow with your needs.

# SetUp & Installation
**just follow these steps:**
- `git clone https://github.com/Ali-googoojani/blBot.git`
- open the repo dir in vscode
- Type `npm i`
- Type `npm run build`
- Creates `blBot folder`
- Create your project folder next to `blBot folder`
- Import lib
- Write your own code

**for more click <a href="https://github.com/Ali-googoojani/blBot/blob/main/docs.md">here</a>** 


# Attention

> ⚠️ **Note: Since this library is not yet published on npm, please import it locally using the path: `import { ... } from './blBot';`**
